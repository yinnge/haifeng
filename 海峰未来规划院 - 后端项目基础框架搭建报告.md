三、关键配置说明

Spring Security 白名单

"/api/v1/*/auth/login"
"/api/v1/*/auth/register"
"/api/v1/*/auth/refresh"
"/actuator/**"

MyBatis-Plus 配置

- ID生成策略：ASSIGN_ID（雪花算法）
- 逻辑删除字段：isDeleted
- 自动填充：createdAt / updatedAt

Redis Key 规范

haifeng:{模块}:{标识}
例：haifeng:token:refresh:{userId}

五、启动说明

# 1. 启动数据库
docker-compose up -d

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填写数据库密码、JWT密钥等

# 3. 启动用户端
./mvnw spring-boot:run -pl haifeng-app

# 4. 验证健康检查
curl http://localhost:8080/actuator/health
